# Changelog

## v0.1.0

Initial public release of WBSync.

### Added

- WBS creation with parent-child task structure
- Ticket management with Kanban-style status columns
- Timeline view for WBS-based Gantt charting
- CSV export for WBS, tickets, and timeline data
- Local JSON project storage
- Windows one-file EXE build workflow
- GitHub-ready documentation, issue templates, and marketing copy
