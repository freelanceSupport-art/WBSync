# 使い方

## WBSを作成する

1. `WBS Studio` を開きます。
2. `追加` でWBSを作成します。
3. 親子構造を作りたい場合は、親WBSを選んで `子を追加` を押します。
4. 開始日、終了日、担当、状態、進捗率を入力します。

## チケットを管理する

1. `Ticket Board` を開きます。
2. `追加` でチケットを作成します。
3. 状態、優先度、期限、担当者を入力します。
4. 必要に応じてWBSへ紐付けます。

## ガントチャートを確認する

1. `Timeline` を開きます。
2. 表示単位を `日`、`週`、`月` から選びます。
3. WBSの期間と進捗を横断して確認します。

## CSVへ出力する

上部の `CSV出力` から出力先フォルダを選ぶと、WBS、チケット、ガント用CSVをまとめて生成します。

## データ保存

アプリは `wbsync_project.json` にプロジェクトデータを保存します。EXEで起動した場合は、EXEと同じフォルダに保存されます。
