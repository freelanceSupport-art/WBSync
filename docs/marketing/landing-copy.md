# WBSync 紹介文

## One-Liner

WBSyncは、WBS、チケット管理、ガントチャートをひとつにまとめたWindows向けローカルプロジェクト管理ツールです。

## Short Copy

ExcelでWBSを作り、別のツールで課題を管理し、さらにガントを別管理していると、進捗確認だけで時間が溶けていきます。  
WBSyncは、工程、チケット、スケジュールをひとつの画面構成にまとめ、ローカル環境で素早く管理できるようにしたデスクトップアプリです。

## Value Props

- WBSとチケットを同じプロジェクトデータで管理できます。
- ガントチャートで期間と進捗をすぐ確認できます。
- CSV出力で既存の報告フローへつなげられます。
- 単体EXEとして配布でき、Python環境がないPCでも使えます。

## GitHub Release Description

WBSyncは、WBS作成、チケット管理、ガントチャート、CSV出力をまとめたWindows向けデスクトップツールです。

このリリースでは以下を利用できます。

- WBSの親子階層管理
- チケットのKanban表示
- WBSベースのガントチャート
- WBS、チケット、ガントのCSV出力
- 単体EXEでの起動

`WBSync.exe` をダウンロードして起動してください。プロジェクトデータはEXEと同じフォルダに保存されます。
